The file 'osd-tshirt-no-logo.svg' is the vectory expanded text version of the tshirt design for 2024 and 2025 where correct OSD expanded colours (beyond white and black) are used. When getting this tshirt printed, please add the OSD logo in a location and size that is proportional to the photo 'photo of shirt with osd logo and a cat'. You can find the open source design logo in the file 'logo-white.jpg'.

This tshirt design much be printed on black and/or a very dark colour so the white text and white logo is legible.

We recommend that you get the tshirts printed locally using a print company that you trust and support the business of. Local companies that use sustainable practices and materials are preferred.

If ordered a bulk order of tshirts please limit your order to approx $300.00 and place an expense request in our open collective example: https://opencollective.com/opensourcedesign/expenses/237862 

Please submit a [pull request](https://github.com/opensourcedesign/merch-swag/pulls) with evidence of the printed tshirts and where/what event you intend to distribute them. This means your expense request is more likely to be approved.
